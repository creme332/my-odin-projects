#  🛠 Installation
Clone repository locally.
```bash
git clone git@github.com:creme332/my-odin-projects.git
```
Move to project.
```bash
 cd my-odin-projects/restaurant-page
 ```

 Install dependencies.
 ```bash
npm install
 ```
